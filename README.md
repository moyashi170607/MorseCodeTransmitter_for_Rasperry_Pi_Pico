# モールス信号プロジェクト

このプロジェクトは、Raspberry Piを使用してモールス信号を学習および練習するためのものです。

## 特徴

- テキストをモールス信号にエンコード
- モールス信号の速度を調整可能
- モールス信号の視覚および音声表現
